1. Download input files from: https://datasets.imdbws.com/
	title.akas.tsv.gz 
	title.ratings.tsv.gz

2. Put input files in the same directory as final.py

3. Run the program. You might need to wait a couple minutes before the output files are generated. The program is complete after best_10_releases_IMDB.txt and worst_10_releases_IMDB.txt are fully populated with 10 releases respectively.
